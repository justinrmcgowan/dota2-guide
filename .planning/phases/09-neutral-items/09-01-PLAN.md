    Section header: ## Neutral Items
    Preamble: When the user message includes a "Neutral Items Catalog" section, apply these principles:
    5 numbered rules:
    1. **Rank by hero synergy:** pick 2-3 per tier matching hero kit/role/matchup. Explain WHY.

    4. **No strong preference is acceptable:** do not force-rank when difference is negligible.
    5. **Tier timing awareness:** T1=5min, T2=15min, T3=25min, T4=35min, T5=60min. Consider hero timing.

    Add rule 10 to Output Constraints: neutral_items field is array of tier objects. Each has "tier" (int 1-5) and "items" array of 2-3 picks. Each pick has "item_name" (string), "reasoning" (string, 1 sentence), "rank" (int, 1=best). Only include tiers in catalog. If no catalog, return [].
