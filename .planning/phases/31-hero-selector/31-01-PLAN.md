2. **_get_bracket_matrices(cache, bracket)** helper:
```python
def _get_bracket_matrices(cache: DataCache, bracket: int) -> tuple[list, list, dict[int, int]]:
    bracket_data = cache._matrices.get(bracket, {})
    raw_mapping = bracket_data.get("hero_id_to_index", {})
    hero_id_to_index = {int(k): v for k, v in raw_mapping.items()}
    synergy = bracket_data.get("synergy", [])
    counter = bracket_data.get("counter", [])
    return synergy, counter, hero_id_to_index
```

3. **score_candidates(candidate_ids, ally_ids, enemy_ids, synergy, counter, hero_id_to_index)** function:
```python
def score_candidates(...) -> dict[int, tuple[float, float, float]]:
    # returns {hero_id: (composite, synergy_score, counter_score)}
```
- If synergy or counter is empty, return {hid: (0.0, 0.0, 0.0) for hid in candidate_ids}
- For each candidate: look up c_idx = hero_id_to_index.get(hid); if None, score = 0.0
- syn_vals = [synergy[c_idx][hero_id_to_index[a]] for a in ally_ids if hero_id_to_index.get(a) is not None and synergy[c_idx][hero_id_to_index[a]] is not None]
- ctr_vals = [counter[c_idx][hero_id_to_index[e]] - 0.5 for e in enemy_ids if hero_id_to_index.get(e) is not None and counter[c_idx][hero_id_to_index[e]] is not None]
- syn = sum(syn_vals)/len(syn_vals) if syn_vals else 0.0
- ctr = sum(ctr_vals)/len(ctr_vals) if ctr_vals else 0.0
- composite = syn * 0.4 + ctr * 0.4
- Guard array bounds: wrap index access in try/except IndexError → score 0.0 as final safety net

4. **HeroSelector class**:
```python
class HeroSelector:
    def get_suggestions(self, request: SuggestHeroRequest, cache: DataCache) -> SuggestHeroResponse:
```
- Get all heroes from cache.get_all_heroes()
- Role filter: viable_ids = HERO_ROLE_VIABLE.get(request.role, set()); if len(viable_ids) < 5: viable_ids = {h.id for h in all_heroes}
- Build candidate list: heroes with id in viable_ids and NOT in excluded_hero_ids
- matrices_available: bool = len(cache._matrices) > 0 and bool(cache._matrices.get(request.bracket, {}).get("synergy"))
- Call _get_bracket_matrices + score_candidates
- Sort candidates by composite score descending, then by localized_name ascending (stable secondary sort)
- Build top_n HeroSuggestion objects; round scores to 4 decimal places
- Return SuggestHeroResponse(suggestions=..., matrices_available=matrices_available)
