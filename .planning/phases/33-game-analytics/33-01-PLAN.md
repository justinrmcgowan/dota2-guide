**GET /match-history** (per D-10, D-12):
- Query params: `hero_id: int | None`, `result: str | None` ("win"/"loss"), `mode: str | None` ("fast"/"auto"/"deep"), `limit: int = 50`, `offset: int = 0`
- Query `MatchLog` table ordered by `played_at DESC`
- Apply filters: if hero_id, filter by hero_id; if result="win", filter win=True; if result="loss", filter win=False; if mode, filter engine_mode=mode
- For each match, eager-load related `MatchItem` and `MatchRecommendation` rows using subqueries (two additional queries keyed by match_log_id IN [...ids])
- Return Pydantic response model:

```python
class MatchHistoryItem(BaseModel):
    id: int
    match_id: str
    hero_id: int
    hero_name: str
    role: int
    playstyle: str | None
    side: str | None
    lane: str | None
    win: bool
    duration_seconds: int | None
    kills: int

    assists: int
    gpm: int
    xpm: int
    net_worth: int
    engine_mode: str | None
    was_fallback: bool
    overall_strategy: str | None
    follow_rate: float | None
    played_at: str  # ISO format string
    items: list[MatchItemPayload]
    recommendations: list[MatchRecommendationPayload]

class MatchHistoryResponse(BaseModel):
    matches: list[MatchHistoryItem]
    total: int
    limit: int
    offset: int
```

- First query: SELECT with filters + LIMIT/OFFSET, also run a COUNT query for total
- Second query: batch-load MatchItem rows WHERE match_log_id IN (matched ids)
- Third query: batch-load MatchRecommendation rows WHERE match_log_id IN (matched ids)
- Group items and recommendations by match_log_id and attach to each MatchHistoryItem

**GET /match-stats** (per D-13, D-14, D-15):
- No query params (aggregates over all matches)
- Single query computing:
  - `total_games`: COUNT(*)
  - `wins`: COUNT(*) WHERE win=True
  - `win_rate`: wins / total_games (0 if no games)
  - `avg_follow_rate`: AVG(follow_rate) WHERE follow_rate IS NOT NULL
  - `avg_follow_rate_wins`: AVG(follow_rate) WHERE win=True AND follow_rate IS NOT NULL
  - `avg_follow_rate_losses`: AVG(follow_rate) WHERE win=False AND follow_rate IS NOT NULL
- Return:

```python
class MatchStatsResponse(BaseModel):

    wins: int
    losses: int
    win_rate: float
    avg_follow_rate: float | None
    avg_follow_rate_wins: float | None
    avg_follow_rate_losses: float | None
```

Use SQLAlchemy `select`, `func.count`, `func.avg` with the async session pattern from `get_db`. All queries use `await session.execute(stmt)`.
