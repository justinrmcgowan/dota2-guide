Edit prismlab/frontend/nginx.conf to add a new location /admin/ block AFTER the existing location /api/ block.

The admin router in main.py is mounted with app.include_router(admin_router) (no prefix), meaning the backend endpoint is at /admin/refresh-data on port 8000.

Add this location block after the existing /api/ block:

    # Proxy admin requests to backend container
    location /admin/ {
        proxy_pass http://prismlab-backend:8000/admin/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

This mirrors the existing /api/ location block pattern exactly.
  </action>
