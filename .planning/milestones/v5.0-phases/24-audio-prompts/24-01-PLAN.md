Create `prismlab/frontend/src/stores/audioStore.ts`:

```typescript
import { create } from "zustand";
import { persist } from "zustand/middleware";

interface AudioStore {
  enabled: boolean;
  volume: number; // 0.0 to 1.0
  setEnabled: (v: boolean) => void;
  setVolume: (v: number) => void;
}

export const useAudioStore = create<AudioStore>()(
  persist(
    (set) => ({
      enabled: true,
      volume: 0.7,
      setEnabled: (enabled) => set({ enabled }),
      setVolume: (volume) => set({ volume: Math.max(0, Math.min(1, volume)) }),
    }),
    { name: "prismlab-audio", version: 1 },
  ),
);
```

Create `prismlab/frontend/src/utils/audioUtils.ts` with three exports:

1. `sanitizeForSpeech(msg: string): string` — strips dash-dash, clock notation, converts gold notation:
   - `.replace(/--/g, ",")` strips " -- " separators
   - `.replace(/\(\d+:\d+\)/g, "")` removes "(35:00)" clock notation
   - `.replace(/\+(\d+)g/, "$1 gold")` converts "+2000g" to "2000 gold"
   - `.trim()`

2. `humanizeItemName(name: string): string` — converts snake_case item names to Title Case for speech:
   - `.replace(/_/g, " ")` replaces underscores with spaces
   - `.replace(/\b\w/g, (c) => c.toUpperCase())` capitalizes first letter of each word
   - "blade_mail" → "Blade Mail", "black_king_bar" → "Black King Bar"

3. `speak(text: string, volume: number): void` — wraps `window.speechSynthesis.speak()`:
   - Guard: `if (!("speechSynthesis" in window)) return;`
   - Cancel pending speech first: `window.speechSynthesis.cancel();`
   - Create `new SpeechSynthesisUtterance(text)` with:
     - `utterance.volume = volume`
     - `utterance.rate = 1.1` (slightly faster for urgency)
     - `utterance.pitch = 1.0`
     - Do NOT set `utterance.voice` (use browser default to avoid getVoices() async issue)
   - Chrome 15-second keepalive: in `utterance.onstart`, start a `setInterval` at 10,000ms that calls `window.speechSynthesis.pause()` then `window.speechSynthesis.resume()` if `window.speechSynthesis.speaking` is true, otherwise clears itself. Store the interval ID in a local `let keepaliveTimer` variable.
   - In `utterance.onend` and `utterance.onerror`: clear the keepalive interval.
   - Call `window.speechSynthesis.speak(utterance)`.
