Create `prismlab/frontend/src/hooks/useAudio.ts` with a single exported function `useAudio(): void`.

The hook has three useEffect blocks:

**Block 1: AudioContext unlock for autoplay policy**
```typescript
useEffect(() => {
  const unlock = () => {
    if (!audioCtxRef.current) {
      audioCtxRef.current = new AudioContext();
    } else if (audioCtxRef.current.state === "suspended") {
      audioCtxRef.current.resume().catch(() => {});
    }
  };
  document.addEventListener("click", unlock);
  return () => document.removeEventListener("click", unlock);
}, []);
```
Use `const audioCtxRef = useRef<AudioContext | null>(null);` declared before the effects.

**Block 2: Toast subscription (GSI trigger events)**
Subscribe to `useRefreshStore` and speak toasts when audio is enabled:
```typescript
useEffect(() => {
  return useRefreshStore.subscribe((state) => {
    const { enabled, volume } = useAudioStore.getState();
    if (!enabled || !state.lastToast) return;
    speak(sanitizeForSpeech(state.lastToast.message), volume);
  });
}, []);
```
Use `useRefreshStore.getState()` style via subscribe (not a React hook call — subscribe is the Zustand store subscription API that returns an unsubscribe function).

**Block 3: Recommendation subscription (new recommendations)**
Subscribe to `useRecommendationStore` and announce the top priority item when a new recommendation arrives:
```typescript
useEffect(() => {
  const prevDataRef = { current: null as RecommendResponse | null };
  return useRecommendationStore.subscribe((state) => {
    const { enabled, volume } = useAudioStore.getState();
    if (!enabled) return;
    if (state.data && state.data !== prevDataRef.current && !state.isLoading) {
      prevDataRef.current = state.data;
      const topItem = state.data.phases[0]?.items[0];
      if (topItem) {
        speak(`Consider buying ${humanizeItemName(topItem.item_name)}`, volume);
      }
    }
  });
}, []);
```

Important implementation notes:
- Import `useRef` from "react". Import `useEffect` from "react".
- Import `useRefreshStore` from "../stores/refreshStore"
- Import `useRecommendationStore` from "../stores/recommendationStore"
- Import `useAudioStore` from "../stores/audioStore"
- Import `speak`, `sanitizeForSpeech`, `humanizeItemName` from "../utils/audioUtils"
- Import `type RecommendResponse` from "../types/recommendation"
- The subscribe callback in Block 2 uses `.getState()` to get the current audio settings — this avoids stale closure captures of enabled/volume.
- `useAudioStore.getState()` (not `useAudioStore()`) is used inside subscribe callbacks — never call React hooks inside non-hook callbacks.
