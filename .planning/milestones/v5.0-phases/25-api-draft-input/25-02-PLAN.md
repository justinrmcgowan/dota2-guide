**1. Create `prismlab/frontend/src/utils/steamId.ts`** — Steam ID conversion per research Pattern 3.

```typescript
const STEAM_ID_64_BASE = 76561197960265728n;

export function steamId64ToAccountId(steamId64: string): number {
  const id64 = BigInt(steamId64);
  return Number(id64 - STEAM_ID_64_BASE);
}

export function accountIdToSteamId64(accountId: number): string {
  return String(STEAM_ID_64_BASE + BigInt(accountId));
}

export function isValidSteamId(input: string): boolean {
  try {
    if (!/^\d{17}$/.test(input)) return false;  // Must be 17 digits
    const accountId = steamId64ToAccountId(input);
    return accountId > 0 && accountId < 2 ** 32;
  } catch {
    return false;
  }
}
```

**1. Create `prismlab/frontend/src/hooks/useLiveDraft.ts`** — the core live draft polling hook.

This hook:
- Exposes `fetchDraft()` for manual refresh (D-05) and `isPolling` state for UI
- On GSI connect, immediately fetches draft + starts 10s polling (D-03, D-04)
- On GSI disconnect, stops polling
- Stops polling when all 10 heroes are picked
- Populates gameStore with allies, opponents, side, and optionally role from Stratz position data

```typescript
import { useCallback, useEffect, useRef, useState } from "react";
import { useGsiStore } from "../stores/gsiStore";
import { useGameStore } from "../stores/gameStore";
import { api } from "../api/client";
import { steamId64ToAccountId } from "../utils/steamId";
import type { Hero } from "../types/hero";
import type { LiveMatchResponse } from "../types/livematch";
```

Export `function useLiveDraft(heroes: Hero[]): { fetchDraft: () => void; isPolling: boolean }`.

Internal state:
- `pollRef = useRef<ReturnType<typeof setInterval> | null>(null)` for polling timer
- `draftCompleteRef = useRef(false)` to track when all 10 heroes picked
- `[isPolling, setIsPolling] = useState(false)` for UI feedback
- `heroesRef = useRef(heroes)` kept in sync via useEffect

`processDraftData` function (called by fetchDraft):
- Takes `match: LiveMatchResponse, accountId: number, heroList: Hero[]`
- Find user in players by account_id === accountId
- If user not found, return (game not found for this player -- per Pitfall 4)
- Determine myTeamIsRadiant from user's is_radiant
- Set side: `useGameStore.getState().setSide(myTeamIsRadiant ? "radiant" : "dire")`
- Filter allies (same team, exclude self, hero_id > 0) and opponents (other team, hero_id > 0) -- per Pitfall 2, skip unpicked heroes (hero_id === 0)
- Map hero_ids to Hero objects using heroList.find(h => h.id === player.hero_id)
- Auto-fill gameStore (D-06, D-08 -- overwrite existing):
  - `allies.forEach((hero, i) => { if (hero) gameStore.setAlly(i, hero); })` -- fill up to 4 ally slots
  - `opponents.forEach((hero, i) => { if (hero) gameStore.setOpponent(i, hero); })` -- fill up to 5 opponent slots
- Auto-set user's own hero: find user's hero_id in heroList, call `gameStore.selectHero(hero)` if found and hero_id > 0
- Auto-suggest role from Stratz position data (per Pitfall 5): if user's position is 1-5, call `gameStore.setRole(position)`. Only if position is a valid number 1-5. Do NOT override if position is null/undefined (OpenDota has no position data).
- Check if draft is complete: `match.players.every(p => p.hero_id > 0)`. If so, set draftCompleteRef.current = true.

`fetchDraft` callback:
- Read steamId from localStorage key "prismlab_steam_id"
- If no steamId, return
- Convert to accountId via steamId64ToAccountId
- Call `api.getLiveMatch(accountId)` -- wrap in try/catch, log errors, return on failure
- If no match returned, return (no live match found)
- Call processDraftData with the result

GSI subscription useEffect:
- Subscribe to gsiStore. On transition to "connected" (prev gsiStatus !== "connected" and current === "connected"):
  - Reset draftCompleteRef to false
  - Call fetchDraft() immediately
  - Start polling: `pollRef.current = setInterval(() => { if (!draftCompleteRef.current) fetchDraft(); }, 10_000)` (D-04: every 10s)
  - setIsPolling(true)
- On transition away from "connected":
  - Clear interval, set pollRef.current = null
  - setIsPolling(false)
- Cleanup: unsubscribe + clear interval

Return `{ fetchDraft, isPolling }`.
