**prismlab/backend/data/models.py** -- SQLAlchemy 2.0 ORM models:
Use Mapped and mapped_column syntax (SQLAlchemy 2.0 style).

Hero model (__tablename__ = "heroes"):
- id: Mapped[int] = mapped_column(Integer, primary_key=True) -- Dota 2 hero ID
- name: Mapped[str] = mapped_column(String, nullable=False) -- display name
- localized_name: Mapped[str] = mapped_column(String, nullable=False) -- e.g. "Anti-Mage"
- internal_name: Mapped[str] = mapped_column(String, nullable=False) -- e.g. "npc_dota_hero_antimage"
- primary_attr: Mapped[str | None] = mapped_column(String) -- "agi", "str", "int", or "all"
- attack_type: Mapped[str | None] = mapped_column(String) -- "Melee" or "Ranged"
- roles: Mapped[list | None] = mapped_column(JSON) -- e.g. ["Carry", "Escape"]
- Float stats: base_health, base_mana, base_armor, base_str, base_agi, base_int, str_gain, agi_gain, int_gain
- Integer stats: base_attack_min, base_attack_max, attack_range, move_speed
- img_url: Mapped[str | None] = mapped_column(String) -- CDN portrait URL
- icon_url: Mapped[str | None] = mapped_column(String) -- CDN icon URL
- updated_at: Mapped[datetime | None] = mapped_column(DateTime, default=func.now())

Item model (__tablename__ = "items"):
- id: Mapped[int] = mapped_column(Integer, primary_key=True)
- name: Mapped[str] = mapped_column(String, nullable=False) -- display name like "Black King Bar"
- internal_name: Mapped[str] = mapped_column(String, nullable=False) -- like "blink"
- cost: Mapped[int | None] = mapped_column(Integer, nullable=True)
- components: Mapped[list | None] = mapped_column(JSON, nullable=True)
- is_recipe: Mapped[bool] = mapped_column(Boolean, default=False)
- is_neutral: Mapped[bool] = mapped_column(Boolean, default=False)
- tier: Mapped[int | None] = mapped_column(Integer, nullable=True)
- bonuses: Mapped[dict | None] = mapped_column(JSON, nullable=True)
- active_desc: Mapped[str | None] = mapped_column(Text, nullable=True)
- passive_desc: Mapped[str | None] = mapped_column(Text, nullable=True)
- category: Mapped[str | None] = mapped_column(String, nullable=True)
- tags: Mapped[list | None] = mapped_column(JSON, nullable=True)
- img_url: Mapped[str | None] = mapped_column(String)
- updated_at: Mapped[datetime | None] = mapped_column(DateTime, default=func.now())

MatchupData model (__tablename__ = "matchup_data"):
- id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)
- hero_id: Mapped[int] = mapped_column(Integer, ForeignKey("heroes.id"))
- opponent_id: Mapped[int] = mapped_column(Integer, ForeignKey("heroes.id"))
- win_rate: Mapped[float | None] = mapped_column(Float)
- games_played: Mapped[int | None] = mapped_column(Integer)
- common_items: Mapped[list | None] = mapped_column(JSON)
- common_starting_items: Mapped[list | None] = mapped_column(JSON)
- bracket: Mapped[str] = mapped_column(String, default="high")
- updated_at: Mapped[datetime | None] = mapped_column(DateTime, default=func.now())

**prismlab/backend/data/opendota_client.py:**
Create an httpx-based async client for the OpenDota API.
- Class OpenDotaClient with __init__(self, api_key: str | None = None) that stores params dict: self.params = {"api_key": api_key} if api_key else {}
- async def fetch_heroes(self) -> dict: GET https://api.opendota.com/api/constants/heroes with 30s timeout, return .json() -- returns dict keyed by hero ID strings like {"1": {...}, "2": {...}}
- async def fetch_items(self) -> dict: GET https://api.opendota.com/api/constants/items with 30s timeout, return .json() -- returns dict keyed by item internal names like {"blink": {...}}
- Both methods use httpx.AsyncClient() context manager and call response.raise_for_status()

**prismlab/backend/data/seed.py:**
Create the auto-seed function called during FastAPI lifespan startup.

CRITICAL PITFALL: OpenDota returns keyed objects, NOT arrays. Must iterate with .items().

Function async def seed_if_empty():
1. Open async session via async with async_session() as session:
2. Check count: await session.scalar(select(func.count()).select_from(Hero)) -- if > 0, return immediately
3. Create OpenDotaClient(api_key=settings.opendota_api_key)
4. Hero seeding: fetch heroes, iterate heroes_data.items(). For each hero:
   - Extract slug: info["name"].replace("npc_dota_hero_", "")
   - Create Hero with id=info["id"], name=info["localized_name"], localized_name=info["localized_name"], internal_name=info["name"], primary_attr=info.get("primary_attr", "all"), attack_type=info.get("attack_type", "Melee"), roles=info.get("roles", []), and all base stats with defaults
   - Set img_url=f"https://cdn.cloudflare.steamstatic.com/apps/dota2/images/dota_react/heroes/{slug}.png"
   - Set icon_url=f"https://cdn.cloudflare.steamstatic.com/apps/dota2/images/dota_react/heroes/icons/{slug}.png"
5. Item seeding: fetch items, iterate items_data.items(). For each item:
   - Skip entries with no "id" field (malformed)
   - Create Item with id=info["id"], name=info.get("dname", internal_name), internal_name=internal_name, cost=info.get("cost")
   - Detect recipes: is_recipe=internal_name.startswith("recipe_")
   - Detect neutrals: is_neutral=info.get("qual") == "rare"
   - Set img_url=f"https://cdn.cloudflare.steamstatic.com/apps/dota2/images/dota_react/items/{internal_name}.png"
   - Map bonuses from info.get("attrib"), active_desc from info.get("hint") first element if list, passive_desc from info.get("lore")
6. await session.commit()
7. Print seeded count

Imports: from sqlalchemy import select, func; from data.database import async_session; from data.models import Hero, Item; from data.opendota_client import OpenDotaClient; from config import settings
