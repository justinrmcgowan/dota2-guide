    - `setSide(side)` -- sets side
    - `setLane(lane)` -- sets lane
    - `toggleLaneOpponent(hero)` -- if hero.id already in laneOpponents, remove it. Otherwise, add it only if laneOpponents.length < 2
    - `clearLaneOpponents()` -- sets laneOpponents to []
