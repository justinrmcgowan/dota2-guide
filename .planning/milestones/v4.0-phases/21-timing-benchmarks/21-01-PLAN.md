```python
@dataclass(frozen=True)
class TimingBucket:
    time: int          # seconds since game start
    games: int
    wins: int
    confidence: str    # "strong" | "moderate" | "weak"

    @property
    def win_rate(self) -> float:
        return self.wins / self.games if self.games > 0 else 0.0

class DataCache:
    def get_hero_timings(self, hero_id: int) -> dict[str, list[TimingBucket]] | None:
        """Returns dict of {item_internal_name: [TimingBucket, ...]} sorted by time."""
        return self._timing_benchmarks.get(hero_id)

    def get_item_validation_map(self) -> dict[int, tuple[int | None, str]]:
        """Return {item_id: (cost, internal_name)} for all items."""

    def hero_id_to_name(self, hero_id: int) -> str:
        """Lookup hero localized_name by ID."""
```

    1. `classify_timing_zones(buckets: list[TimingBucket]) -> dict[str, Any] | None`:
       - Return None if `not buckets`, `len(buckets) < 2`, or `sum(b.games for b in buckets) < 50`
       - Compute `weighted_avg_wr = sum(b.win_rate * b.games) / total_games`
       - Compute `peak_wr = max(b.win_rate for b in buckets)`
       - Classify each bucket: good if `win_rate >= peak_wr - 0.02`, ontrack if `win_rate >= weighted_avg_wr`, else late
       - Compute aggregate win rates for good/late zones (weighted by games)
       - Urgency: `is_urgent = (good_avg_wr - late_avg_wr) > 0.10` (per 21-RESEARCH.md, Claude's discretion)
       - Confidence: "weak" if any bucket is weak, "moderate" if any is moderate, else "strong"
       - Return dict with keys: `good_range`, `ontrack_range`, `late_range` (pre-formatted minute strings), `good_win_rate`, `late_win_rate` (0-1 floats, rounded to 3 decimals), `is_urgent` (bool), `confidence` (str), `total_games` (int), `buckets_classified` (list of dicts with time/games/win_rate/confidence/zone)

    2. `_format_time_range(buckets: list[TimingBucket]) -> str`:
       - Convert seconds to minutes (divide by 60)
       - Single bucket: "~{min} min"
       - Multiple buckets: first bucket is min_time, last is max_time
       - For the good zone (lowest times): "< {max_time} min"
       - For the late zone (highest times): "> {min_time} min"
       - For ontrack zone (middle): "{min_time}-{max_time} min"

    CRITICAL: TimingBucket.time is in SECONDS, not minutes. All display strings must convert to minutes.
